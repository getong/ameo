Open Developer tools

Run this in the Browser Dev console

        $ameo.subscribe("topic1");

Run this in a shell

        redis-cli publish topic1 "hi there"

You should see a message arriving in the Dev console

Run this in the Browser Dev console

        $ameo.unsubscribe("topic1");

Run this in a shell

        redis-cli publish topic1 "hi there, again"

You should NOT see a message arriving in the Dev console

You can also ping the server

        $ameo.ping();
